# Recobro Next.js PT

Prueba técnica de Next.js para Recobro: Dashboard SaaS multi-tenant, con separación clara entre Server y Client Components y arquitectura escalable.

## Estructura inicial
- app/ : rutas y páginas
- components/ : Client Components
- lib/ : datos mock / fetch logic
- services/ : lógica de negocio / repositorios
- types/ : tipos TypeScript
